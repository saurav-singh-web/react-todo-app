# React To-Do App

## Description

This is a simple To-Do List application built using React and Vite.
It allows users to add, edit, delete, and mark tasks as completed.

---

## Features

* Add new tasks with priority (High, Medium, Low)
* Delete tasks
* Mark tasks as completed
* Edit existing tasks
* Clean and responsive user interface

---

## Technologies Used

* React (Functional Components & Hooks)
* Vite
* CSS

---

##  How to Run the Project

1. Clone the repository:

```
git clone https://github.com/saurav-singh-web/react-todo-app.git
```

2. Navigate to the project folder:

```
cd react-todo-app
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```

---

## Project Structure

* App.jsx → Manages state and logic
* ToDoList.jsx → Handles form and list rendering
* ToDoItem.jsx → Displays individual tasks
* Header.jsx → Displays app title
